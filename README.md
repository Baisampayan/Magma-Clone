This project is a fully responsive and interactive **clone of the ThisIsMagma.com website**, developed using modern web technologies like **HTML**, **CSS**, **JavaScript**, and powerful animation libraries such as **GSAP**, **Locomotive Scroll**, and **ScrollTrigger**.
